# Technical Support Guide

## Account Access

### I forgot my password. How do I reset it?

To reset your password:
1. Go to the login page
2. Click "Forgot Password?" below the login form
3. Enter your registered email address
4. Check your email for a password reset link (check spam folder if needed)
5. Click the link and create a new password
6. Password must be at least 8 characters with one number and one special character

The reset link expires after 24 hours. If expired, request a new one.

### My account is locked. What should I do?

Accounts are automatically locked after 5 failed login attempts for security. To unlock:
1. Wait 30 minutes for automatic unlock, OR
2. Click "Unlock My Account" on the login page and follow email instructions
3. If urgent, contact support with your account email

For security, we cannot unlock accounts over chat or phone without verification.

### How do I enable two-factor authentication?

Enable 2FA for enhanced security:
1. Go to "Settings" > "Security"
2. Click "Enable Two-Factor Authentication"
3. Scan the QR code with your authenticator app (Google Authenticator, Authy, etc.)
4. Enter the 6-digit code from your app to confirm
5. Save your backup codes in a secure location

We strongly recommend enabling 2FA to protect your account.

### I lost access to my 2FA device. How do I recover my account?

If you've lost your 2FA device:
1. Use one of your backup codes (provided during 2FA setup)
2. If you don't have backup codes, click "Lost 2FA Device?" on login page
3. Verify identity through email confirmation
4. Provide government-issued ID if requested
5. Support will disable 2FA so you can log in and set up a new device

Account recovery may take 24-48 hours for security verification.

### I'm not receiving verification emails. What should I check?

If you're not getting our emails:
1. Check your spam/junk folder
2. Add noreply@example.com to your contacts/whitelist
3. Verify your email address is correct in account settings
4. Check if your inbox is full
5. Try adding an alternate email address
6. Wait 5-10 minutes (sometimes there's a delay)
7. Contact your email provider to ensure emails aren't blocked

If none of these work, contact support from an alternate email address.

### Can I log in from multiple devices simultaneously?

Yes! You can be logged in on:
- Up to 5 devices simultaneously on all plans
- Enterprise plans: unlimited devices
- You can view active sessions in Settings > Security > Active Sessions
- You can remotely log out of any session for security

If you see unfamiliar devices, log them out immediately and change your password.

### The website isn't loading properly. What should I try?

Try these troubleshooting steps:

1. **Clear browser cache**:
   - Chrome: Ctrl+Shift+Delete (Windows) or Cmd+Shift+Delete (Mac)
   - Select "Cached images and files"
   - Click "Clear data"

2. **Disable browser extensions**:
   - Ad blockers can interfere with functionality
   - Try incognito/private mode to test

3. **Update your browser**:
   - Check for and install any available updates

4. **Check JavaScript**:
   - Ensure JavaScript is enabled in browser settings

5. **Try a different browser**:
   - Rules out browser-specific issues

If issues persist after these steps, contact support with:
- Your browser name and version
- Screenshot of the error
- Description of what you're trying to do

### Features aren't working on mobile. How can I fix this?

**Mobile Browser**:

## Connection & Performance

### The service is slow. How can I improve performance?

To improve speed:

**Check Your Internet**:
1. Run speed test at speedtest.net
2. We require minimum 5 Mbps download speed
3. Recommended: 25 Mbps+ for optimal experience

**Optimize Your Connection**:
1. Use wired Ethernet instead of Wi-Fi when possible
2. Move closer to Wi-Fi router
3. Restart your router (unplug for 30 seconds)
4. Close bandwidth-heavy applications (streaming, downloads)
5. Disable VPN temporarily to test

**Browser Optimization**:
1. Close unnecessary browser tabs
2. Clear browser cache
3. Disable resource-heavy extensions
4. Update to latest browser version

**Check Our Status**:
- Visit status.example.com for system status
- Subscribe to status updates

If slow after these steps, contact support with speed test results.

### I'm getting error messages. What do they mean?

Common error codes:

**Error 500 - Internal Server Error**:
- Our server encountered an issue
- Try again in 5-10 minutes
- If persistent, contact support

**Error 404 - Page Not Found**:
- URL is incorrect or page was moved
- Check URL for typos
- Try searching for the content
- Contact support if you believe it's an error

**Error 403 - Access Denied**:
- You don't have permission to access this resource
- Log out and log back in
- Check your subscription level
- Contact support if you should have access

**Error 401 - Unauthorized**:
- Session expired - log in again
- Clear cookies and try again

**Connection Timeout**:
- Check your internet connection
- Disable firewall/antivirus temporarily to test
- Try different network (mobile hotspot)

**Service Unavailable**:
- Server is down for maintenance
- Check status.example.com
- Usually resolved within 30 minutes

Always screenshot error messages before contacting support.

### Files aren't uploading. What are the limits?

**File Upload Limits**:
- Basic: 25MB per file
- Pro: 100MB per file
- Enterprise: 500MB per file

**Supported File Types**:
- Documents: PDF, DOC, DOCX, TXT, RTF
- Images: JPG, JPEG, PNG, GIF, BMP
- Spreadsheets: XLS, XLSX, CSV
- Presentations: PPT, PPTX
- Archives: ZIP (max 200MB)

**Upload Troubleshooting**:
1. Check file size and type
2. Ensure stable internet (don't switch networks during upload)
3. Try different browser
4. Temporarily disable firewall/antivirus
5. Rename file (remove special characters)
6. Compress file if too large

**Upload Best Practices**:
- Upload during off-peak hours for large files
- Use wired connection for files over 50MB
- Don't close browser until upload completes
- Wait for confirmation message

### Why am I getting logged out frequently?

Auto-logout occurs for security:

**Session Timeouts**:
- Idle timeout: 60 minutes of inactivity
- Maximum session: 12 hours
- Option to "Remember Me" extends to 30 days

**To Stay Logged In**:
1. Check "Keep me logged in" at login
2. Enable cookies for example.com
3. Whitelist our domain in ad blockers
4. Use browser's password manager

**Security Note**:
- Logging out after inactivity protects your account
- Adjust timeout in Settings > Security > Session Timeout
- Don't use "Remember Me" on shared computers

### The app keeps crashing. How do I fix it?

Try these steps in order:

1. **Force close the app**:
   - iOS: Swipe up, swipe app away
   - Android: Settings > Apps > Force Stop

2. **Restart your device**:
   - Complete power off and on

3. **Update the app**:
   - Check App Store/Play Store for updates

4. **Clear app cache**:
   - Android: Settings > Apps > Example > Clear Cache
   - iOS: Delete and reinstall (iOS doesn't have cache clear)

5. **Free up storage**:
   - Need at least 500MB free space
   - Delete unused apps or photos

6. **Reinstall the app**:
   - Delete app completely
   - Restart device
   - Reinstall from store

7. **Check device compatibility**:
   - iOS 13+ or Android 8+ required
   - 2GB RAM minimum

**If Still Crashing**:
- Contact support with:
  - Device model and OS version
  - When crashes occur (specific action?)
  - Screenshot of error if shown

## Data & Security

To download your data:

1. Log in to your account
2. Go to Settings > Privacy > Data Export
3. Select data types to export:
   - Account information
   - Messages
   - Files
   - Settings
   - Activity logs
4. Click "Request Data Export"
5. Receive email with download link (within 48 hours)
6. Download expires after 7 days

**Export Formats**:
- Account data: JSON
- Messages: JSON and CSV
- Files: Original formats in ZIP

**Data Includes**:
- All your personal information
- Complete message history
- Uploaded files
- Account settings
- Activity logs

### How do I permanently delete my account and data?

To delete your account:

1. Go to Settings > Privacy > Delete Account
2. Read deletion consequences carefully
3. Enter your password to confirm
4. Click "Permanently Delete My Account"
5. Confirm via email link

**Important Notes**:

**Alternative**:
Consider "Deactivate Account" instead:
- Temporarily disables account
- Data preserved
- Can reactivate anytime within 90 days
